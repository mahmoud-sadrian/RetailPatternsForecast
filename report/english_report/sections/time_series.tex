\section{Time Series Forecasting (Manual SES)}

A Simple Exponential Smoothing model was implemented manually. The recursion is:
\[
s_t = \alpha\, y_t + (1-\alpha)\, s_{t-1},
\]
and the forecast is:
\[
\hat{y}_{t+h} = s_t.
\]
